## 프로젝트 소개

### One Bit Class

One Bit Class는 모든 회원이 수강생이 될 수 있고, 강사도 될 수 있는 플랫폼을 제공합니다.   

누구나 강사가 되고, 수강생이 될 수 있기 때문에 회원가입을 한 회원들은 누구든 강좌를 개설할 수 있습니다.   
그리고 모든 회원은 다른 사람이 개설한 강좌를 수강할 수 있습니다.   

이 뿐만 아니라 회원가입을 하지 않은 비회원이라도 어떤 강좌가 개설됐는지   
확인할 수 있도록 제공하고 있습니다.
